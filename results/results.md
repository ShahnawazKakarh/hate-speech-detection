# Results

> Last updated: 2026-06-15. Generated from `artifacts/<run>/metrics.json` via `scripts/make_results_table.py`. Cross-dataset numbers from `scripts/cross_dataset_eval.py`.

## In-domain test set

Hate as positive class. All models trained with seed 42, identical preprocessing, default 0.5 decision threshold.

| Model | Dataset | Acc | P | R | F1 (hate) | F1 (macro) | AUC |
|---|---|---|---|---|---|---|---|
| TF-IDF + LR  | Davidson   | 0.9044 | 0.3150 | 0.5594 | 0.4030 | 0.6755 | 0.8566 |
| TF-IDF + LR  | HateXplain | 0.8280 | 0.7064 | 0.7576 | 0.7311 | 0.8023 | 0.8917 |
| Doc2Vec + LR | Davidson   | 0.5547 | 0.0938 | 0.7762 | 0.1674 | 0.4317 | 0.7218 |
| Doc2Vec + LR | HateXplain | 0.6996 | 0.5089 | 0.7694 | 0.6126 | 0.6836 | 0.7955 |
| DistilBERT   | Davidson   | 0.9391 | 0.4583 | 0.3077 | 0.3682 | 0.6681 | 0.8966 |
| DistilBERT   | HateXplain | **0.8503** | **0.7593** | 0.7542 | **0.7568** | **0.8243** | **0.9088** |

**Headline take-aways**

- **DistilBERT wins HateXplain across the board** — F1-hate 0.757, macro-F1 0.824, AUC 0.909. The class-balanced corpus lets contextual modelling shine.
- **Davidson is dominated by class imbalance** (5.8% hate). At the default threshold, DistilBERT predicts conservatively and TF-IDF + class-balanced LR ekes out a higher F1-hate (0.40 vs 0.37). DistilBERT still wins on AUC (0.897 vs 0.857), meaning the ranking is better — what we need is threshold tuning, not a stronger model. Covered in the paper's discussion + ablations.
- **Doc2Vec underperforms on Davidson** (F1-hate 0.17) because ~1,150 hate-class training examples is not enough to learn a useful 200-dim representation. On HateXplain (~4,700 hate examples) it lands at a respectable F1-hate 0.61.

## Cross-dataset generalization

Trained on one corpus, evaluated on the other corpus's test split. Tests how brittle each model is to a distribution shift in writing style, topic, and label definition.

| Model | Train → Eval | F1 (hate) | F1 (macro) | AUC | Δ vs in-domain (macro-F1) |
|---|---|---|---|---|---|
| TF-IDF + LR  | Davidson → HateXplain | 0.4837 | 0.5870 | 0.6584 | **−0.215** |
| TF-IDF + LR  | HateXplain → Davidson | 0.2693 | 0.6070 | 0.6890 | **−0.069** |
| Doc2Vec + LR | Davidson → HateXplain | 0.4736 | 0.4265 | 0.5697 | **−0.257** |
| Doc2Vec + LR | HateXplain → Davidson | 0.1390 | 0.4539 | 0.5941 | **−0.230** |
| DistilBERT   | Davidson → HateXplain | 0.3702 | 0.5871 | 0.6890 | **−0.237** |
| DistilBERT   | HateXplain → Davidson | 0.2593 | **0.6128** | **0.8118** | **−0.055** |

**Headline take-aways**

- **Every model degrades** going out-of-domain — expected, but the size of the drop varies a lot. The honest reading is that none of these models would deploy cleanly on a new platform without re-tuning.
- **DistilBERT generalizes best from HateXplain → Davidson** (AUC 0.812, the highest cross-dataset AUC), suggesting contextual representations are more transferable than n-gram or static-embedding features when the source corpus has enough class signal.
- **Davidson is the harder target**: the extreme imbalance hits every cross-trained model. Macro-F1 holds up because the non-hate class dominates, but F1-hate collapses.

## Reproduction

Every number above is regenerated by:

```bash
python -m hsd.data.prepare --dataset all
for cfg in configs/*.yaml; do python -m hsd.train --config "$cfg"; done
python scripts/cross_dataset_eval.py
python scripts/make_results_table.py
```

Seed: 42. Splits: Davidson is 80/10/10 stratified by 3-way label, HateXplain uses the official split. Decision threshold: 0.5 (un-tuned). Hardware reported in `paper/main.tex`.
